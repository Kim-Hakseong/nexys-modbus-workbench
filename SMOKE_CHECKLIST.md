# 스모크 체크리스트 — Nexys Modbus Workbench

배포 전 Windows PC에서 실제 exe로 확인한다. **물리 장비가 없으면 내장 시뮬레이터
([시뮬레이터...] 버튼, 127.0.0.1:1502)로 0~6장 전체를 수행할 수 있다.**

## 0. 기본

- [ ] `Nmw.App.exe` 더블클릭 실행 — 설치/런타임 없이 메인 윈도우가 뜬다
- [ ] 창 크기 조절/최대화 시 그리드·로그 레이아웃이 깨지지 않는다

## 1. 연결 (F-01, F-02, F-17)

- [ ] [연결...] → Modbus TCP: IP/포트 입력 → 연결 성공 시 상태바에 `TCP host:port` 표시
- [ ] 잘못된 IP로 연결 → "연결 실패" + 원인 메시지, 앱은 정상 동작 유지
- [ ] RTU: COM 포트 목록이 뜨고 보레이트/패리티/비트 설정 가능 (USB-RS485 장착 시)
- [ ] 타임아웃/재시도/프레임간 지연 값이 통신에 반영된다 (타임아웃 짧게 → Timeout 에러 확인)
- [ ] [해제] → 상태 "연결 안 됨", 모든 폴 정지

## 2. 폴링 (F-03, F-05, F-07)

- [ ] FC03, 주소 0, 개수 10, 500ms → 시작: 값 갱신, Tx/OK 카운트 증가
- [ ] Scan rate 10ms에서 UI 프리즈 없음
- [ ] FC01 코일 폴 → ON/OFF 표시
- [ ] 폴 정지 → 카운트 멈춤, 재시작 → 이어서 증가
- [ ] 폴 탭 2개 이상 동시 폴링 (동일 채널 공유, 시리얼에서도 응답 섞임 없음)

## 3. 쓰기 (F-04)

- [ ] [쓰기...] FC06: 값 쓰기 → "성공 (n ms)" → 폴 그리드에 새 값 반영
- [ ] FC16: `10, 0x0102` 목록 쓰기 → 재독 일치
- [ ] FC05/FC15 코일 쓰기 → FC01 폴로 확인
- [ ] 없는 주소에 쓰기 → "실패: Illegal Data Address" 표시
- [ ] 그리드 값 더블클릭 → 쓰기 다이얼로그에 주소/현재값 프리필

## 4. 포맷/주소 (F-06, F-10)

- [ ] Float32 + 워드오더 4종 변경 시 값 해석이 바뀐다 (파이 3.1415927 벡터 등)
- [ ] S16 음수, Hex16, Bin16 표시 확인
- [ ] [PLC 주소(1-base)] 토글 → 그리드 주소가 40001 스타일로, 입력 주소도 자동 변환

## 5. 에러 처리/트래픽 로그 (F-08, F-09, F-18)

- [ ] 장비 전원 차단/케이블 분리 → Timeout 에러 카운트 증가, 에러 명칭 표시
- [ ] 트래픽 로그에 TX/RX raw hex 프레임 + 타임스탬프 표시
- [ ] [에러만] 필터 → ERR 라인만, [클리어] → 비움
- [ ] [저장...] → .txt 파일 내용 = 화면 로그
- [ ] 장시간(수 시간) 폴링에도 로그 5000라인 유지, 메모리 증가 없음

## 6. 워크스페이스 (F-11)

- [ ] [저장...] → `.nmw` 저장 (JSON, 한글 별칭이 그대로 보임)
- [ ] 앱 재시작 → [열기...] → 연결 설정/폴 탭/별칭/주소 표기 복원 + 자동 연결
- [ ] 손상된 `.nmw` 열기 → 명확한 오류 메시지, 기존 상태 유지

## 7. 내장 시뮬레이터 (장비 없는 테스트)

- [ ] [시뮬레이터...] → 시작 → 상태에 "수신 중 — 127.0.0.1:1502" 표시
- [ ] 메인 창에서 TCP 127.0.0.1:1502 연결 → 폴링 성공, 클라이언트/요청 카운트 증가
- [ ] 시뮬레이터 그리드에서 홀딩 값 입력 → 폴 그리드에 반영
- [ ] [값 자동 변화] ON → 폴 값이 계속 증가
- [ ] 마스터에서 FC06/16/05/15 쓰기 → 시뮬레이터 그리드에 반영
- [ ] 주소 1000 이상 폴링 → "Illegal Data Address" 표시
- [ ] 시뮬레이터 정지 → 마스터에 Timeout/연결 오류 표시, 재시작 → 재연결 가능

## 8. 배포 (F-12)

- [ ] exe 단일 파일 크기 ≤ 90MB
- [ ] .NET 미설치 PC(Windows 10 1809+)에서 실행 확인
